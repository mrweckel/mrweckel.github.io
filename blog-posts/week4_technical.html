Enumerable#group_by

August 27, 2014

You can use the group_by on a enumerable object to return a hash on a block. The key becomes the result hash and the value of the key is an array that matches the key. Lets look at an example:

names = ["Matt", "Jen", "James", "Alfred", "John", "Dan", "Mark", "Don"]

puts names.group_by {|name| name.length}

This would return:

{4=>["Matt", "John", "Mark"], 3=>["Jen", "Dan", "Don"], 5=>["James"], 6=>["Alfred"]}

This is an incredibly useful way to quickly make a hash that groups items according to some kind of shared characteristic. I might just go use this on challenge 2.
